# Pendulum Simulation

This is a simulation of a double pendulum

## Running the project

Make sure you have python installed. To run the project:

```shell
$ python double_pendulum.py
```

Make sure you have `python-tk` intalled. On macOS, you can install it like this:

```
$ brew install python-tk
```

## Controls

- Press space to pause/play simulation
- Press 'r' to reset the simulation
- Click and drag the weights to change their positions and bar lengths
- Use the sliders to change the Gravitational Constant and Drag Coefficient
